# 尺寸标注线绘制分析

## 概述

`plant3d-web` 项目中的尺寸标注系统位于 `src/utils/three/annotation/` 目录，提供了专业的 CAD 风格三维标注功能。本文档重点分析尺寸标注线的绘制机制。

## 核心类结构

### 1. LinearDimension3D（主要实现）

**文件位置**: `src/utils/three/annotation/annotations/LinearDimension3D.ts`

这是三维线性尺寸标注的核心实现类，采用 **SolveSpace 风格**的绘制方式。

#### 组件构成

尺寸标注由以下子组件组成：

```typescript
// 尺寸线（可能被文字留白切分为多段）
- dimensionLineA: Line2          // 尺寸线第一段
- dimensionLineB: Line2           // 尺寸线第二段（当文字在中间时）
- dimensionLineOutside: Line2     // 尺寸线外部延伸段（当文字在外部时）

// 尺寸界线（从测量点到尺寸线）
- extensionLine1: Line2           // 起点界线
- extensionLine2: Line2           // 终点界线

// 箭头（V 形线段）
- arrow1: LineSegments2           // 起点箭头
- arrow2: LineSegments2           // 终点箭头

// 文字标签
- textLabel: SolveSpaceBillboardVectorText  // 矢量文字（Billboard 朝向相机）

// 交互辅助
- snapGuideLine: THREE.Line       // 吸附提示线（拖拽文字时显示）
- snapMarkers: THREE.Mesh[]       // 吸附点标记（0, 0.25, 0.5, 0.75, 1 位置）
```

#### 绘制流程

##### 1. 初始化阶段（`constructor`）

```typescript
constructor(materials, params, options) {
  // 1. 创建几何体
  this.dimLineGeometryA = new LineGeometry()
  this.dimLineGeometryB = new LineGeometry()

  // 3. 创建箭头（LineSegments2）
  this.arrow1 = new LineSegments2(this.arrowGeometry1, this.materialSet.line)
  this.arrow2 = new LineSegments2(this.arrowGeometry2, this.materialSet.line)

```

##### 2. 几何重建阶段（`rebuild()`）

```typescript
private rebuild(): void {
  // 1. 计算偏移方向（垂直于 start-end 方向）
  if (this.params.direction) {
    this.offsetDir.copy(this.params.direction).normalize()
  } else {
    // 自动计算：在 XY 平面内垂直
    this.tempVec.copy(end).sub(start).normalize()
    this.offsetDir.set(-this.tempVec.y, this.tempVec.x, 0)
    this.offsetDir.normalize()
  }
  
  // 2. 计算尺寸线端点（局部坐标）
  this.dimStart.copy(start).addScaledVector(this.offsetDir, offset)
  this.dimEnd.copy(end).addScaledVector(this.offsetDir, offset)
  
  // 3. 设置占位几何（真正的像素对齐在 update() 中完成）
  this.dimLineGeometryA.setPositions([
    this.dimStart.x, this.dimStart.y, this.dimStart.z,
    this.dimEnd.x, this.dimEnd.y, this.dimEnd.z,
  ])
  
  // 4. 更新文字内容
  const distance = start.distanceTo(end)
  this.textLabel.setText(`${distance.toFixed(decimals)}${unit}`)
  
  // 5. 设置文字位置（优先 labelOffsetWorld，否则 labelT 插值）
  const baseLabelPos = this.dimStart.clone().lerp(this.dimEnd, labelT)
  this.textLabel.object3d.position.copy(baseLabelPos)
}
```

##### 3. 每帧更新阶段（`update(camera)` → `_updateSolveSpaceGeometry()`）

这是**最关键的绘制逻辑**，每帧都会执行，实现 SolveSpace 风格的像素级精确绘制：

```typescript
private _updateSolveSpaceGeometry(camera: THREE.Camera): void {
  // 1. 获取视口分辨率
  const res = this.materialSet.line.resolution
  const vw = Math.max(1, Math.floor(res?.x ?? 1))
  const vh = Math.max(1, Math.floor(res?.y ?? 1))
  
  // 2. 计算世界空间每像素对应的世界单位（wpp）
  const wpp = worldPerPixelAt(camera, refWorld, vw, vh)
  
  // 3. 获取相机显示轴（SolveSpace: projRight/projUp）
  this.camRight.set(1, 0, 0).applyQuaternion(camera.quaternion).normalize()
  this.camUp.set(0, 1, 0).applyQuaternion(camera.quaternion).normalize()
  
  // 4. 计算尺寸线端点（世界坐标）
  const aeW = this.localToWorld(this.dimStart)  // 尺寸线起点（世界）
  const beW = this.localToWorld(this.dimEnd)    // 尺寸线终点（世界）
  const startW = this.localToWorld(this.params.start)  // 测量起点
  const endW = this.localToWorld(this.params.end)      // 测量终点
  
  // 5. 绘制尺寸界线（超出箭头 10px）
  const ext = 10 * wpp  // 10 像素对应的世界单位
  this.setLineGeometryFromWorld(
    this.ext1Geometry,
    startW,
    aeW.addScaledVector(outDirW, ext),  // 超出 10px
    camera, vw, vh
  )
  
  // 6. 计算文字包围盒（用于留白切分）
  const extPx = this.textLabel.getExtentsPx()
  const swidth = (extPx.width + 8) * wpp   // 文字宽度 + 8px 内边距
  const sheight = (extPx.height + 8) * wpp
  
  // 7. 尺寸线留白切分（文字位置留空）
  const trim = lineTrimmedAgainstBoxT(
    refWorld,      // 文字中心（世界坐标）
    aeW, beW,     // 尺寸线端点
    camRight, camUp,  // 显示轴
    swidth, sheight  // 文字包围盒尺寸
  )
  
  // 8. 绘制尺寸线分段
  // trim.segmentsT 可能是：
  // - [[0, 0.3], [0.7, 1]]  // 文字在中间，切分为两段
  // - [[0, 1]]              // 文字在外部，整段绘制
  // - []                     // 文字覆盖整条线，不绘制
  
  for (const [t0, t1] of trim.segmentsT) {
    const p0 = aeW.clone().addScaledVector(dlW, t0)
    const p1 = aeW.clone().addScaledVector(dlW, t1)
    this.setLineGeometryFromWorld(geom, p0, p1, camera, vw, vh)
  }
  
  // 9. 绘制箭头（V 形，13px 宽，18° 半角）
  const theta = 18 * Math.PI / 180
  const width = 13 * wpp
  const legLen = width / Math.cos(theta)
  
  // 箭头方向：如果文字在外部，箭头指向外部
  const arrowDir = trim.within !== 0 ? -dirUnit : dirUnit
  
  // 构建 V 形箭头几何（两条线段）
  const tip = aeW  // 箭头尖端
  const e1 = arrowDir.clone().applyAxisAngle(normal, +theta).setLength(legLen).add(tip)
  const e2 = arrowDir.clone().applyAxisAngle(normal, -theta).setLength(legLen).add(tip)
  
  // 10. 像素对齐（所有端点）
  alignToPixelGrid(camera, tip, vw, vh, alignedTip)
  alignToPixelGrid(camera, e1, vw, vh, alignedE1)
  alignToPixelGrid(camera, e2, vw, vh, alignedE2)
  
  // 11. 更新几何体
  this.arrowGeometry1.setPositions([
    alignedTip.x, alignedTip.y, alignedTip.z,
    alignedE1.x, alignedE1.y, alignedE1.z,
    alignedTip.x, alignedTip.y, alignedTip.z,
    alignedE2.x, alignedE2.y, alignedE2.z,
  ])
  
  // 12. 参考尺寸虚线处理
  if (this.params.isReference) {
    // 虚线参数：4px dash，4px gap
    material.dashSize = 4 * wpp
    material.gapSize = 4 * wpp
    material.dashed = true
    line.computeLineDistances()  // 需要计算距离用于虚线渲染
  }
}
```

#### 关键技术点

##### 1. 像素对齐（`alignToPixelGrid`）

```typescript
// 所有线段端点都进行像素对齐，避免模糊
export function alignToPixelGrid(
  camera: THREE.Camera,
  worldPoint: THREE.Vector3,
  viewportWidthPx: number,
  viewportHeightPx: number
): THREE.Vector3 {
  // 1. 投影到 NDC（标准化设备坐标）
  const ndc = worldPoint.project(camera)
  
  // 2. 转换为屏幕像素坐标
  const sx = (ndc.x * 0.5 + 0.5) * viewportWidthPx
  const sy = (-ndc.y * 0.5 + 0.5) * viewportHeightPx
  
  // 3. 四舍五入到整数像素
  const sxR = Math.round(sx)
  const syR = Math.round(sy)
  
  // 4. 转换回 NDC
  ndc.x = (sxR / viewportWidthPx) * 2 - 1
  ndc.y = -((syR / viewportHeightPx) * 2 - 1)
  
  // 5. 反投影回世界坐标（保持深度 z）
  return ndc.unproject(camera)
}
```

**作用**: 确保线条在屏幕上渲染时对齐到像素网格，避免抗锯齿导致的模糊。

##### 2. 世界单位到像素转换（`worldPerPixelAt`）

```typescript
// 计算在指定世界点位置，1 像素对应的世界单位距离
export function worldPerPixelAt(
  camera: THREE.Camera,
  worldPoint: THREE.Vector3,
  viewportWidthPx: number,
  viewportHeightPx: number
): number {
  // 1. 投影到 NDC
  const ndc = worldPoint.project(camera)
  
  // 2. 计算 +1px 在 X 和 Y 方向对应的世界距离
  const p0 = ndc.unproject(camera)
  const p1 = (ndc + [2/w, 0, 0]).unproject(camera)  // +1px X
  const p2 = (ndc + [0, 2/h, 0]).unproject(camera)  // +1px Y
  
  // 3. 取平均值
  const dx = p0.distanceTo(p1)
  const dy = p0.distanceTo(p2)
  return (dx + dy) * 0.5
}
```

**作用**: 将 SolveSpace 风格的像素尺寸（如 13px 箭头宽度）转换为世界单位，确保在不同相机距离下保持相同的屏幕尺寸。

##### 3. 尺寸线留白切分（`lineTrimmedAgainstBoxT`）

```typescript
// 当文字标签位于尺寸线上时，在文字位置留出空白
export function lineTrimmedAgainstBoxT(
  ref: THREE.Vector3,        // 文字中心
  a: THREE.Vector3,          // 尺寸线起点
  b: THREE.Vector3,          // 尺寸线终点
  gr: THREE.Vector3,         // 显示右轴
  gu: THREE.Vector3,         // 显示上轴
  swidth: number,            // 文字包围盒宽度（世界单位）
  sheight: number            // 文字包围盒高度（世界单位）
): SolveSpaceTrimResultT {
  // 1. 构建文字包围盒的 4 个平面（对齐显示轴）
  const planes = [
    { n: gu * -1, d: -(gu·ref + sheight/2) },  // 上平面
    { n: gu, d: gu·ref - sheight/2 },            // 下平面
    { n: gr, d: gr·ref - swidth/2 },            // 右平面
    { n: gr * -1, d: -(gr·ref + swidth/2) },    // 左平面
  ]
  
  // 2. 计算线段与包围盒的交点
  // 线段参数方程: p(t) = a + (b-a)*t, t ∈ [0,1]
  
  // 3. 根据交点位置返回分段
  // - 如果文字在中间: segmentsT = [[0, t0], [t1, 1]]
  // - 如果文字在外部: segmentsT = [[0, 1]], within = ±1
  // - 如果文字覆盖整条线: segmentsT = []
}
```

**作用**: 当文字标签位于尺寸线上时，自动在文字位置留出空白，避免文字与线条重叠。

##### 4. 箭头绘制（V 形线段）

```typescript
// SolveSpace 风格箭头：V 形，13px 宽，18° 半角
const theta = 18 * Math.PI / 180        // 半角
const width = 13 * wpp                   // 箭头宽度（世界单位）
const legLen = width / Math.cos(theta)   // 箭头腿长度

// 箭头方向
const arrowDir = trim.within !== 0 ? -dirUnit : dirUnit

// 构建 V 形（两条线段）
const tip = aeW                          // 箭头尖端
const e1 = arrowDir.applyAxisAngle(normal, +theta).setLength(legLen).add(tip)
const e2 = arrowDir.applyAxisAngle(normal, -theta).setLength(legLen).add(tip)

// 像素对齐后设置几何
arrowGeometry.setPositions([
  tip.x, tip.y, tip.z, e1.x, e1.y, e1.z,  // 第一条腿
  tip.x, tip.y, tip.z, e2.x, e2.y, e2.z,  // 第二条腿
])
```

**特点**:
- 使用 `LineSegments2` 绘制两条线段形成 V 形
- 箭头宽度固定为 13 像素（屏幕空间）
- 半角 18°，符合 CAD 标准

### 2. LinearDimension（简化版本）

**文件位置**: `src/utils/three/annotation/annotations/LinearDimension.ts`

这是简化版本的线性尺寸标注，使用：
- `CSS2DObject` 作为文字标签（而非矢量文字）
- `THREE.Mesh` 作为箭头（圆锥体，而非 V 形线段）
- 不进行像素对齐和留白切分

**适用场景**: 简单的 2D 或 3D 标注，不需要 SolveSpace 风格的精确像素控制。

## 绘制管线总结

```
1. 初始化（constructor）
   ├─ 创建几何体（LineGeometry, LineSegmentsGeometry）
   ├─ 创建 Line2/LineSegments2 对象
   ├─ 创建文字标签
   └─ 调用 rebuild() 设置初始几何

2. 几何重建（rebuild）
   ├─ 计算偏移方向（垂直于测量方向）
   ├─ 计算尺寸线端点（局部坐标）
   ├─ 设置占位几何
   └─ 更新文字内容和位置

3. 每帧更新（update → _updateSolveSpaceGeometry）
   ├─ 获取视口分辨率
   ├─ 计算世界单位/像素比例（wpp）
   ├─ 获取相机显示轴
   ├─ 转换到世界坐标
   ├─ 绘制尺寸界线（超出 10px）
   ├─ 计算文字包围盒
   ├─ 尺寸线留白切分
   ├─ 绘制尺寸线分段
   ├─ 绘制箭头（V 形，13px，18°）
   ├─ 像素对齐所有端点
   └─ 更新几何体

4. 渲染（Three.js 自动）
   ├─ LineMaterial 处理线宽、虚线
   ├─ 像素对齐确保清晰
   └─ 深度测试控制遮挡关系
```

## 关键技术特性

### 1. SolveSpace 风格

- **像素对齐**: 所有端点对齐到像素网格
- **固定屏幕尺寸**: 箭头、界线超出量等以像素为单位
- **留白切分**: 文字位置自动留空
- **V 形箭头**: 使用线段而非填充形状

### 2. 缩放独立（Scale Independent）

通过 `AnnotationBase.scaleIndependentOfZoom()` 实现：
- 标注在屏幕上保持恒定尺寸
- 不随相机缩放而变化
- 通过计算世界单位/像素比例实现

### 3. 交互状态

- **normal**: 默认状态（洋红色）
- **hovered**: 悬停状态（黄色）
- **selected**: 选中状态（红色）

### 4. 参考尺寸（Reference Dimension）

- 使用虚线样式（4px dash，4px gap）
- 文字后缀 " REF"
- 仅显示，不参与约束

## 相关工具函数

### `solvespaceLike.ts`

提供 SolveSpace 风格的辅助函数：
- `alignToPixelGrid()`: 像素对齐
- `worldPerPixelAt()`: 世界单位/像素转换
- `lineTrimmedAgainstBoxT()`: 尺寸线留白切分

### `AnnotationMaterials.ts`

材质管理系统：
- 预定义颜色集（green, orange, blue, white, yellow）
- SolveSpace 风格颜色（magenta, hovered yellow, selected red）
- 每帧更新分辨率（`setResolution()`）

## 使用示例

```typescript
import { LinearDimension3D } from '@/utils/three/annotation'
import { AnnotationMaterials } from '@/utils/three/annotation'

const materials = new AnnotationMaterials()

const dimension = new LinearDimension3D(
  materials,
  {
    start: new THREE.Vector3(0, 0, 0),
    end: new THREE.Vector3(10, 0, 0),
    offset: 2,              // 尺寸线偏移 2 单位
    labelT: 0.5,           // 文字在中间
    unit: 'mm',
    decimals: 1,
  }
)

scene.add(dimension)

// 每帧更新
function animate() {
  dimension.update(camera)
  renderer.render(scene, camera)
}
```

## 总结

`plant3d-web` 的尺寸标注线绘制系统采用了 **SolveSpace 风格**的专业 CAD 标注方式：

1. **精确的像素控制**: 通过像素对齐和世界单位/像素转换，确保在不同相机距离下保持一致的屏幕尺寸
2. **智能的留白处理**: 自动在文字位置留出空白，避免重叠
3. **专业的视觉效果**: V 形箭头、固定像素尺寸、像素对齐等特性，提供清晰的标注显示
4. **灵活的交互**: 支持悬停、选中、拖拽等交互状态

整个系统设计精良，代码结构清晰，是高质量的三维标注实现。
