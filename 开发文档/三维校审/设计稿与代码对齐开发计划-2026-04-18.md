## 2. review-reviewer.pen 差异清单

### 2.1 ReviewPanel — AnnotationWorkspace 布局与功能差异（高优）

| 维度 | 设计稿 (pen) | 代码实现 (Vue) | 差异等级 |
| --- | --- | --- | --- |
| 批注队列标题 | `当前批注（5）/ 待处理2 / 高优1` | 仅显示 `批注 N / 测量 N` | **功能缺失** |
| ActiveAnnotationCard | 独立高亮卡片展示当前选中批注 | 展开/收起列表项 | 交互差异 |
| FilterRow | 批注筛选行（按类型/状态/优先级） | 无筛选 | **功能缺失** |
| PriorityCard | 批注优先级卡片 | 批注无优先级字段 | **数据模型缺失** |
| CompactRuleHint | 规则提示条 `"选中构件后再打标注"` | 无 | UI 缺失 |
| DraftInput | 专用草稿输入框 | 确认备注仅在确认操作区出现 | 交互差异 |
| ThreadPreview | 评论线程预览 inline 展示 | 展开批注卡片后才显示 CommentsTimeline | 交互差异 |

**决策点**：批注队列/优先级/筛选属于 M8+ 产品化能力，还是当前阶段需补齐？

## 4. 优先级与行动方案
