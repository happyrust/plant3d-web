# Ptset (连接点) 可视化

## 1. Identity

- **What it is:** 为 3D 模型中的元件添加连接点（Point Set）可视化功能。
- **Purpose:** 使用户能够在 3D 场景中以交互式方式查看和定位元件的连接点信息（位置、方向、管道外径等）。

## 2. High-Level Description

Ptset 可视化模块提供了完整的连接点数据加载和渲染系统。用户可以通过模型树的右键菜单选择"显示点集"来触发该功能。系统会从后端 REST API 获取指定元件的点集数据，在 3D 场景中用绿色球体标记每个连接点位置，用橙色箭头表示方向向量，并用屏幕标签显示点编号、坐标和管道外径等信息。该功能支持世界坐标变换矩阵应用、相机实时追踪标签位置更新，并能够自动飞行到点集视图。
