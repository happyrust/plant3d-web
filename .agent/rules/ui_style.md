# UI 样式与设计规则 (Plant3D Web)

在绘制、调整或生成与 `plant3d-web` 相关的用户界面与组件面板时，请**始终**查阅并遵循以下风格和样式约定（源自 `pipe-distance-annotation-panel.pen`）：

1. **核心色彩体系**：
   - 面板背景 (Background): `#FFFFFF` (纯白)
   - 边框与分割线 (Border): `#E5E7EB` (浅灰)
   - 默认文本内容 (Primary Text): `#111827` (深灰/黑)
   - 弱化文本/注释 (Muted Text): `#6B7280` (中等灰度)
   - 品牌/强调交互色 (Primary/Accent Color): `#FF6B00` (亮橙，用于按钮、激活项)
   - 品牌/强调交互底色 (Light Accent): `#FFF0E6` (用于高亮背景、Tag 背景)

2. **字体与排版**：
   - 界面常规说明、模块标题、标签采用：`Fira Sans` 字体系列。
   - 数据展示、数值型标量、ID，如 Refno (`17496_123456`)、距离 (`5000`, `1.2m`)、坐标等，**强制使用**等宽字体：`Fira Code`。
   - 这可以确保关键数据的数值部分在结构上稳定对齐，易于用户阅读和比对。

3. **视觉层次与元素 (Tokens)**：
   - **阴影 (Shadows)**：用于浮动容器（如主面板或下拉框）上以示层级，不要过于沉重。
   - **圆角 (Corner Radius)**：卡片、输入框、按钮等建议保持在 `6px`（较小控件）~ `12px` （或 `16px` 大面板）之间。

4. **交互组件状态**：
   - 输入框 (Inputs)：背景色选用带微透明度或非常浅的颜色（如 `#FFFFFF08` 于暗色背景下或纯白配灰色边框）。使用描边来包裹输入框边缘。
   - 主按钮 (Primary Button)：背景必须强调并明确为可用状态，采用 `#FF6B00` 填充以作引导。
   - 无操作图标：对于纯展示的 Icon，保持与 Muted 文字相同的颜色 `[#6B7280]` 即可。

如果当前在 Pencil 画板进行批量设计 (`batch_design`)，应先**使用 `get_variables`** 设置和获取当前对应的变量或使用硬编码实现上述规则，确保风格完全对齐。
